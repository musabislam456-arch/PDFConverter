# Musab PDF Converter

Musab PDF Converter is a professional all-in-one PDF tools website project built with HTML, CSS, JavaScript, Node.js, and Express.

## Features

- Convert files to PDF
- Merge PDF
- Split PDF
- Compress PDF
- Edit PDF UI
- OCR picture to text
- Image editing tools
- Office-style simple document editor
- Drag and drop upload UI
- Google Drive import button UI
- Responsive for mobile and desktop
- SEO-friendly homepage content
- Privacy and trust section

## Project Files

- `index.html`
- `style.css`
- `script.js`
- `server.js`
- `package.json`
